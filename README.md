# Scoreboard
Simple scoreboard resource that utilises Lation UI to display activities that are available to do in the server if the required police are achieved.
# Dependencies
[lation_ui](https://lationscripts.com/product/modern-ui)
# Installation
- Download the latest `scoreboard`
- Download the latest `lation_ui`
- Drag & drop both resources onto your server
- Make sure `lation_ui` is started before scoreboard
- Restart your server and have fun!
